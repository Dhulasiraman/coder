
# show

 
Show a single user. Use &#39;me&#39; to indicate the currently authenticated user.


## Usage
```console
show &lt;username|user_id|&#39;me&#39;&gt;
```

## Description
```console
  $ coder users show me 
```


## Options
### --output, -o
Output format. Available formats: table, json

| --- | --- |
